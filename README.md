# 🏠 House Price Prediction using Linear Regression

A beginner-friendly Machine Learning project that demonstrates how to build a **Linear Regression model from scratch** using Python, Pandas, NumPy, Matplotlib, Seaborn, and Scikit-Learn.

This project covers the complete Machine Learning workflow including data loading, preprocessing, visualization, model training, evaluation, and prediction.

---

## 📌 Project Overview

The objective of this project is to predict house prices based on several numerical features such as:

- House Size (sq ft)
- Number of Bedrooms
- House Age
- Distance from City
- Number of Bathrooms

The project is designed for beginners who are learning Machine Learning and want to understand how Linear Regression works step-by-step.

---

## 📂 Dataset

This dataset was created from scratch for learning purposes.

### Features

| Feature | Description |
|----------|-------------|
| Size_sqft | House size in square feet |
| Bedrooms | Number of bedrooms |
| Bathrooms | Number of bathrooms |
| Age | Age of the house |
| Distance_City | Distance from city center (km) |
| Price | Target variable |

## 📈 Machine Learning Algorithm

**Linear Regression**

Linear Regression finds the best-fit line that models the relationship between independent variables and the target variable.

Equation:

Price = m₁x₁ + m₂x₂ + ... + b

where

- x = Features
- m = Coefficients
- b = Intercept

---

## 📊 Model Evaluation

The model is evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score
